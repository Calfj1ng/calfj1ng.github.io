# 用 Astro 搭建博客

这个博客是使用 [Astro](https://astro.build) 框架搭建的，部署在 GitHub Pages 上。

## 为什么选择 Astro

Astro 是一个现代的静态站点生成器，适合内容驱动的网站：

- **快速** — 默认零 JavaScript，页面加载极快
- **简单** — 使用 Markdown 写文章，无需复杂配置
- **灵活** — 支持内容集合、组件化开发
- **SEO 友好** — 静态输出，搜索引擎友好

## 技术栈

| 技术 | 用途 |
|------|------|
| Astro | 静态站点生成 |
| Markdown | 博客内容编写 |
| GitHub Pages | 免费托管 |
| GitHub Actions | 自动部署 |

Astro 原生支持 Markdown，你可以方便地使用各种语法：

```javascript
// 代码高亮

  console.log('Hello, Blog!');
```
